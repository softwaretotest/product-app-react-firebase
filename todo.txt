2) Store / Shop / Settings แยก currency ออกไป
shop = {
  id: 1,
  name: "My Shop",
  currency: "USD"
}

หรือจะใช้แบบ settings:

settings = {
  currency: "USD",
  locale: "en-US"
}

แค่เปลี่ยนจากนี้
currency = "THB";
👉 เป็นแบบนี้
store.currency = "THB";

currency = DEFAULT_BY_LANG[lang];

currency = store.currency;

  lang: "th",
  currency: "THB",
  name: "My Shop",
};
